Project 2 Online Retail

Research Question: Can we predict the total value of a single order using only attributes of that transaction (like quantity, items, time, and product)?

The dataset I will be using consits of transactional data of transactions from 01/12/2010 through 09/12/2011 for a UK-based and registered non-store online retail. I want to explore this data with predictive modeling train a linear regression model to see what prediction values I get.

Distribution of Order Value

# Log transformed plot (more readable)

sns.histplot(np.log1p(order_df["order_value"]), kde=True)
plt.title("Log-Transformed Order Value")
plt.show()

What first plot shows

  • The distribution is extremely right-skewed.
  • Most orders are low value, while a small number of invoices have very large totals.
  • This pattern is typical in real-world retail data

Second plot why log-transform?

  • Reduces the impact of extreme outliers
  • Makes the distribution more symmetric
  • Helps linear regression perform better
  • Creates a more stable relationship between predictors and the target variable

Result

After the log transform, the distribution becomes much smoother and closer to a normal shape. This makes it more appropriate to train a regression model that predicts order value.

CONCLUSION

In this project, I explored whether the total value of an order can be predicted using only transaction-level features such as quantity, unit price, product variety, and time of purchase. I found that quantity and the number of unique items are the strongest drivers of order value, while unit price has a smaller effect and temporal features like month and weekday contribute minimally. The distribution of order values is highly right-skewed with many outliers, so I applied a log transformation to improve model stability and interpretability. Using multivariable linear regression, I was able to explain a significant portion of the variability in order value, showing that transaction-level attributes alone provide meaningful predictive signals, although additional information such as customer behavior or promotions would be needed for fully accurate predictions.
